📸 How to add your own sky photos

In the <script> below, find the archive array. Each month has a skies list. For each sky, set src to your image filename (e.g. "photos/bangalore-nov-1.jpg"), give it a location, name, and the colors array (hex codes you pick from the photo). Upload your photos to the same folder as index.html in your GitHub repo. The site will blur them automatically to create the gradient effect — hover reveals the real photo.
